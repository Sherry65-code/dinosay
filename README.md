- How to install

    Copy paste this in your terminal
    
    ```bash
    git clone https://github.com/Sherry65-code/dinosay
    cd dinosay
    chmod a+x ./install.sh
    ./install.sh
    ```

- To run dinosay just type - 

   ```bash
   dinosay <Whatever>
   ```
